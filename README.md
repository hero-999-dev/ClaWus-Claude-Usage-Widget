# ClaWus-Claude-Usage-Widget

What does ClaWus do?

A lightweight Windows system tray app that tells you the best time to use Claude.
It sits in your taskbar as a small colored dot that updates every 30 seconds:

Green — US servers mostly idle, best time to use Claude

Amber — Moderate load, still usable

Red — US peak hours, limits hit faster

Click the dot to open a full weekly schedule in your local timezone, with task-specific recommendations (coding, writing, data work, etc.) and settings for theme, color filter, and time format.

Why it works: 

Claude's servers are hosted in the US. During American working hours, API demand spikes — rate limits kick in faster and responses slow down. During US sleeping hours (roughly 02:00–09:00 ET), load drops significantly. ClaWus maps that pattern so you always know when your quota goes further.

Themes & dot colors: 

ClaWus comes with 6 built-in themes (Dark, Light, Mocha, Macchiato, Frappe, Latte). Each theme uses its own color palette, so the exact shade of the tray dot changes with your theme — but the three-state meaning (best / good / avoid) always stays the same. Accessibility color filters are also available for common color vision deficiencies.

Requirements: Windows 10/11 · Python 3.9+

Install: Download setup.exe from Releases, run it, click Install → Launch.
